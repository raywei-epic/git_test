# Git Stash Practice

这是一个专门用来练习 `git stash` 的小仓库。你可以放心修改、提交、stash、制造冲突，因为它和你的真实工作仓库是分开的。

建议学习顺序：

1. 先看 `lessons/01-status-model.md`，理解工作区、暂存区、提交历史。
2. 再看 `lessons/02-stash-basic.md`，练习 `stash` 的基础命令。
3. 最后看 `lessons/03-real-workflow.md`，模拟工作中临时切分支和处理冲突。
4. 忘记命令时看 `STASH-CHEATSHEET.md`。

练习时最重要的一条规则：

```bash
git status
```

不确定当前发生了什么时，先看状态，再决定下一步。
